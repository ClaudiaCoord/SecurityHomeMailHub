
Шифрование сообщений ключами PGP

Для любого аккаунта возможно создать пару PGP ключей, которыми будут шифровавать, дешифровать и подписываться ваши почтовые сообщения.
Опция "Дешифровать входящие сообщения" применяется в случае использования вами почтового клиента не поддерживающего шифрование PGP. В этом случае принимаемые почтовые сообщения дешифруются и доступны для почтового клиента в расшифррованном виде.

В настройках, в разделах "POP3" и "SMTP", также необходимо включить общие правила шифрования, дешифровки почтовых сообщений. При включенных общих правил шифрования, в случае отправки вами не зашифрованного сообщения, перед отправкой конечному адресату, сообщение будет зашифровано ключами получателя, если они доступны.

Экспорт публичного ключа возможен из главного меню - "Настрока почтовых аккаунтов".
Публичный ключ необходим для взаимного сквозного шифрования почтового трафика между абонентами, вы должны передать собственный публичный ключ участнику переписки, а он свой публичный ключ передать вам. 

Пароль:
Общий пароль на существующие и вновь создаваемые PGP ключи. Установив один раз, не рекомендуется изменять в дальнейшем, иначе часть ключей созданных ранее не будет доступна. (обязательно)

Сервер:
Интернет сервер PGP ключей предназначен для облегчения обмена ключами между пользователями через почтовые системы.
Установка адреса сервера не является обязательной для использования PGP шифрования.
Адрес должен начинаться с префикса "hkp://", например:
- hkp://subkeys.pgp.net
- hkp://pgp.mit.edu
- hkp://pool.sks-keyservers.net
- hkp://zimmermann.mayfirst.org
- hkp://pgp-public-keys@keys.<country code>.pgp.net

Путь:
Путь к папке в которую установлен менеджер PGP ключей GnuPG. Совместимая версия GnuPG для работы является 2.0.0 и более ранние. Более поздние версии имеют несовместимый формат выходных файлов.

Установка GnuPG не является обязательной и используется на ваше усмотрение для манипуляций с PGP ключами.
В дистрибутив включен инсталляционный файл GnuPG версии 2.0.0, установка возможна через одноимённую кнопку.
