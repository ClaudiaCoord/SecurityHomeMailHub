
SSH SOCKS прокси туннель
Маршрутизация сетевого трафика с помощью зашифрованного прокси-туннеля SOCKS4 или SOCKS5 через защищённое SSH соединение.

Таким образом, все ваши приложения, использующие SOCKS4-5 прокси сервер, будут подключаться к серверу SSH, и сервер будет перенаправлять весь трафик в его фактическое место назначения. Ваш интернет-провайдер и другие третьи стороны не смогут проверять ваш трафик и блокировать вам доступ к интернет ресурсам.

Настройка сторонних программ для использования SSH прокси:
Приложение должно поддерживать работу через SOCKS4 или SOCKS5 прокси.
После успешного запуска туннеля SSH SOCKS4-5, необходимо настроить выбранное вами приложение для его использования.
Настройки стороннего приложения:
- SOCKS4 необходимо указать сервер 127.0.0.1 и порт 33114,
- SOCKS5 необходимо указать сервер 127.0.0.1 и порт 33115,
- авторизация для локального использования не предусмотрена.

Если вы регистрировали SSH тунель на сайте которые печатают выходные данные регистрации, скопируйте их и нажмите кнопку "Вставить". Если данные обработаны, внимательно проверьте их, возможно некоторые блоки данных не распознаны или некорректно обработаны.

! В случае, если данные с определенного сервера при вставке не могут быть распознаны полностью,
! пожалуйста, пришлите адрес  сервера, для доработки парсера ответов от серверов.

IP адрес:
Адреса SSH сервера к которому будет производиться подключение, предоставляется провайдером SSH тунеля.

Порт сервера:
Порт SSH сервера, обычно относится к диапазону портов: 22, 143 или порты обозначенные как "Dropbear". Предоставляется провайдером SSH тунеля.

Тип прокси сервера:
Возможный выбор: SOCKS4 или SOCKS5, какой тип поддерживает ваше соединение необходимо уточнить у провайдера SSH тунеля.
Обычно это SOCKS5.

Логин:
Имя учетной записи, предоставляется провайдером SSH тунеля. Как правило заполняется пользователем самостоятельно во время регистрации на сервере провайдера SSH тунелей.

Пароль:
Пароль учетной записи, предоставляется провайдером SSH тунеля. Как правило заполняется пользователем самостоятельно во время регистрации на сервере провайдера SSH тунелей.

Импорт/Экспорт:
Есть возможность загрузить или выгрузить выбранный аккаунт в xml формате.
