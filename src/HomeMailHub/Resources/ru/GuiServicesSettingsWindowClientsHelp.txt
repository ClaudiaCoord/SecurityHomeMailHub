
Максимальное время ожидания для почтовых клиентов:
Время ожидания состояния "CONNECT" в миллисекундах, значение по умолчанию 240000 (240 секунд).
Время задержки при состоянии "SEND" является половиной от указанной величины.

Период проверки почты:
Расписание проверки почты на внешних серверах, указывается в минутах, по умолчанию составляет 1440 минут (24 часа).

Всегда очищать сообщения:
Правило применяется к протоколу IMAP для получения почты с внешних серверов.
Принудительно удаляет с внешнего сервера полученные сообщения.

Всегда добавлять поддельный IP адрес в заголовки сообщения:
Правило действует для протокола SMTP, при отправке почты на внешние сервера.
Добавляет в заголовки сообщения фиктивный IP адреса отправителя, равный адресу последнего прокси сервера, через которого произведено подключение.

Получать внешнюю почту только при отправке сообщений:
Правило отключает расписание получения почты с внешних серверов. Фактическое получение почты произойдет только при появлении письма требующего отправки через внешних почтовых провайдеров.
